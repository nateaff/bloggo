Modify fonts

https://fonts.google.com/

https://stoned.io/web-development/hugo/How-To-Dynamically-Use-Google-Fonts-In-A-Hugo-Website/

  1. Add to head.html
<link href='//fonts.googleapis.com/css?family={{ .Site.Params.fontbody }}' rel='stylesheet' type='text/css'>
<link href='//fonts.googleapis.com/css?family={{ .Site.Params.fontheading }}' rel='stylesheet' type='text/css'>

At end of head.html add

<style>
body {
    font-family: '{{ replace .Site.Params.fontbody "+" " "}}';
}

h1, h2, h3,
h4, h5, h6 {
    font-family: '{{ replace .Site.Params.fontheading "+" " "}}';
}
</style>

Here all heading sizes, h1,h2, etc., are being determiend by a single font. You can also repeat the aboves steps but specify the font for individual heading sizes.

Now we can specify the name of the font in the config in the config.toml.

fontbody = "Tinos"
fontheading = "Lato"
